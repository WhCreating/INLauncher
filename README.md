# INLauncher
Я разрабатываю лаунчер для майнкрафта на python и у меня есть цель создать лаунчер, которым можно будет пользоваться, и он будет удобен для обычных пользователей, которые не понимают в программировании.  
  
Сейчас он очень сырой и нестабильный, но я работаю над этим, это можно объяснить тем, что проект реально масштабный, все лаунчеры, которые я видел, написанные на питоне, это были маленькие эксперименты, которые не могли ничего больше установки и запуска своего майнкрафта, а у меня скины, работа с модификациями, свои нововедения и так далее, которые вряд ли вы видели в других лаунчерах(пока это будущее).

Используемые библиотеки:
- [Minecraft launcher Lib](https://pypi.org/project/minecraft-launcher-lib/)
- [flet(0.23.8)](flet.dev)

Переходник:
